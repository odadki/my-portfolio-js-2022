WEATHERAPP

React, JS, HTML, CSS, Flexbox, Bootstrap

This project is designed with React and uses a weather API. Users can type in any city and receive a five-day forecast instantly.

GO TO PROJECT

PROJECT DETAILS:

  1. Page is organized with visual hierarchy, with the search bar at the top, and the results at the bottom.

  2. Input search bar loads with autofocus on to grab user's attention.

  3. When users enter a location, the current temperature is displayed in a bigger font size to indicate its importance. The time, weather conditions, humidity and wind speed results are also displayed in smaller text.

  4. The five-day forecast is shown below in smaller icons and font sizes with the highs and lows for those particular days.

  5. Page does not automatcially reload, so users can search for different cities back-to-back.

  6. This site is fully responsive to ensure the app looks great on any device.
